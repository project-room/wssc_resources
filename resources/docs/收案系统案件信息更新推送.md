### 收案系统案件信息更新推送

#### 1. 数据推送方式:

收案系统通过 HTTP `POST` 请求推送 `JSON` 数据(utf-8编码)

#### 2. 推送数据字段:

```js
{
    "caseId": ""                        // 卷宗系统 caseid, String
    "unique_code": ""                   // 案件唯一码, String, 形如 GYZC_LA_201801_0023
    "data": { ... }                     // 案件信息, Object, 详细字段见下文
}
```
